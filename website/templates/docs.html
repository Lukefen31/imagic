Installation

Windows

Download the installer from the desktop page after purchase. Run the .exe and follow the setup wizard. imagic will be installed to C:\Program Files\imagic.

Two installer variants are available:

  • Standard — imagic with built-in processing (JPEG, PNG, TIFF, embedded RAW previews)
  • Recommended (+ RawTherapee) — includes RawTherapee for true RAW demosaicing and maximum export quality

macOS

Download the .dmg from your purchase email. Open it and drag imagic to your Applications folder. On first launch, right-click and choose "Open" to bypass Gatekeeper.

Linux

Linux support is coming soon. In the meantime you can install via pip:

pip install imagic

pip install (all platforms)

pip install imagic
imagic              # launch the GUI
imagic --headless   # CLI / headless mode

License Activation

After purchasing, you'll receive a license key by email. On first launch, imagic shows an activation dialog:

  1. Enter the license key from your email
  2. Click Activate
  3. imagic validates the key with our server and stores an activation token locally

An internet connection is required only for the initial activation. After that, imagic works fully offline.

Basic Workflow

imagic follows a simple five-step pipeline:

  1. Import — Point at any folder. imagic scans all photos including subfolders.
  2. Analyse — AI scores every photo on sharpness, exposure, noise, composition, and detail.
  3. Review & Cull — See per-metric breakdowns. Keep or trash each shot with one click.
  4. Edit & Grade — Full editing suite with histogram, exposure, colour grading, sharpening, noise reduction, and presets.
  5. Export — Batch-process all kept photos. Crash-resilient pipeline auto-resumes interrupted exports.

AI Culling

The AI analysis pass scores every photo across five metrics:

  • Sharpness — Edge detail, focus accuracy
  • Exposure — Histogram balance, highlight/shadow clipping
  • Noise — ISO noise levels, grain patterns
  • Composition — Rule of thirds, subject placement, dead space
  • Detail — Texture quality, micro-contrast

Burst and duplicate detection uses perceptual hashing to group visually identical shots and automatically pick the sharpest.

The feedback learning system tracks your keep/trash decisions and edits. Over time, scoring aligns with your personal taste.

Editing & Color Grading

The built-in editor includes:

  • Histogram with RGB channel overlay
  • Exposure, contrast, highlights, shadows, whites, blacks
  • Colour temperature, tint, saturation, vibrance
  • Sharpening and noise reduction
  • 6 editing style presets with live previews
  • 30+ cinematic colour grades (LUTs)
  • AI auto-enhance

RawTherapee Integration

The Desktop + RawTherapee variant includes full RAW demosaicing via RawTherapee's processing engine. This gives you:

  • True RAW → JPEG/TIFF conversion (not embedded preview extraction)
  • Full RawTherapee parameter control via PP3 profiles
  • Scene-aware profile auto-selection (portrait, landscape, low-light, high-contrast)
  • 30+ cinematic colour grades applied at the RAW stage

CLI & Headless Mode

# Analyse a folder
imagic --headless --input /photos/wedding --output /photos/culled

# With RawTherapee processing
imagic --headless --input /photos --output /export --profile balanced

Headless mode runs the full pipeline (import → analyse → cull → export) without the GUI, useful for automation and scripts.

Supported Formats

RAW: CR2, CR3 (Canon), NEF (Nikon), ARW (Sony), RAF (Fujifilm), ORF (Olympus), RW2 (Panasonic), DNG (Adobe), PEF (Pentax)

Standard: JPEG, PNG, TIFF, WebP

Troubleshooting

Activation fails

Check your internet connection and ensure you're entering the key exactly as shown in your purchase email. If the issue persists, email snap@imagic.ink with your order ID.

macOS: "imagic can't be opened"

Right-click the app and select "Open", then click "Open" in the dialog. This only needs to be done once.

RawTherapee not detected

If you installed the Standard variant, RawTherapee is not included. Download the Recommended installer from your purchase email for built-in RawTherapee support.
